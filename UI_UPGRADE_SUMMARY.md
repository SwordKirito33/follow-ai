## 🎯 设计原则

### 1. 字体系统
- **主字体**：Inter (300-900)
- **字重使用**：
  - 标题：`font-black` (900)
  - 副标题：`font-bold` (700)
  - 正文：`font-medium` (500-600)
- **字间距**：负值 letter-spacing 用于大标题

### 2. 视觉层次
- **大标题**：`text-5xl` 到 `text-7xl`
- **中标题**：`text-3xl` 到 `text-4xl`
- **小标题**：`text-xl` 到 `text-2xl`
- **正文**：`text-base` 到 `text-lg`

### 3. 颜色系统
- **主色**：蓝色渐变 (#3b82f6 → #6366f1 → #8b5cf6)
- **文本色**：深灰色 (#0f172a, #111827)
- **背景色**：白色和浅色渐变

### 4. 动画系统
- **缓动函数**：`cubic-bezier(0.4, 0, 0.2, 1)`
- **过渡时间**：0.2s 到 0.4s
- **3D效果**：使用 `transform-style: preserve-3d`
