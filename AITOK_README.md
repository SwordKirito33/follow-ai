## 📋 项目概述

**AITok** 是一个结合了：
- 🎬 **抖音机制** - 短视频流、点赞、评论
- 🤖 **AI 创作工具** - 内置图片/视频生成
- 🏷️ **技能标签系统** - 专业认证和展示
- 💼 **接单系统** - 任务发布和匹配
- 📚 **知识付费** - 课程销售平台

的超级内容平台。

---

## 🎯 核心价值

1. **AI 作品展示** → 内容涨粉
2. **专业标签认证** → 建立个人品牌
3. **接单/卖课程** → 变现
4. **AI 能力升级** → 持续成长

---

## 🛠️ 技术栈

### 前端
- React 19 + TypeScript
- Vite
- Tailwind CSS
- Zustand
- React Router

### 后端
- Supabase (PostgreSQL)
- Supabase Auth
- Supabase Storage
- Supabase Realtime

### 第三方服务
- Stripe (支付)
- OpenAI / Stability AI (图片生成)
- Runway / Pika (视频生成)
- GPT-4 / Claude (AI 助理)

### 部署
- Vercel (前端)
- Supabase (后端)
- Cloudflare (CDN)
