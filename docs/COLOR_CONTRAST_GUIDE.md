# 颜色对比度优化指南

## WCAG 标准要求

### 对比度比率要求
| 文本类型 | WCAG AA | WCAG AAA |
|---------|---------|----------|
| 普通文本 (< 18px) | 4.5:1 | 7:1 |
| 大文本 (≥ 18px 或 14px 粗体) | 3:1 | 4.5:1 |
| UI 组件和图形 | 3:1 | 3:1 |

### 深色主题特殊考虑
- Material Design 2 建议深色背景上的白色文字对比度至少 **15.8:1**
- 深色主题下，纯白色 (#FFFFFF) 可能过于刺眼，建议使用 **#F8FAFC** 或 **#E2E8F0**
- 次要文本建议使用 **#94A3B8** 或更亮的灰色

## Follow.ai 当前颜色方案分析

### 背景色
- 主背景: `#0A0E27` (深蓝黑)
- 次背景: `#1A1F3A` (深紫蓝)
- 卡片背景: `rgba(255, 255, 255, 0.05)` (玻璃态)

### 文本色
- 主文本: `#F8FAFC` (接近白色) - ✅ 与 #0A0E27 对比度约 16:1
- 次文本: `#94A3B8` (灰色) - ⚠️ 与 #0A0E27 对比度约 5.5:1 (勉强通过 AA)
- 灰色文本: `#64748B` - ❌ 与 #0A0E27 对比度约 3.5:1 (不通过)

### 强调色
- 青色: `#06B6D4` - ⚠️ 与深色背景对比度约 6:1 (通过 AA)
- 蓝色: `#3B82F6` - ⚠️ 与深色背景对比度约 4.8:1 (勉强通过)
- 紫色: `#8B5CF6` - ⚠️ 与深色背景对比度约 4.2:1 (勉强通过)
- 绿色: `#10B981` - ✅ 与深色背景对比度约 7:1 (通过 AAA)
- 金色: `#F59E0B` - ✅ 与深色背景对比度约 8:1 (通过 AAA)

## 需要修复的问题

### 1. 低对比度文本
- `text-gray-500` (#6B7280) 在深色背景上对比度不足
- `text-gray-600` (#4B5563) 在深色背景上几乎不可见
- 时间戳等次要信息使用的灰色过暗

### 2. 建议的修复方案
| 原始类 | 问题 | 修复方案 |
|--------|------|----------|
| `text-gray-500` | 对比度 ~3.5:1 | 改为 `text-gray-400` (#9CA3AF) |
| `text-gray-600` | 对比度 ~2.5:1 | 改为 `text-gray-400` (#9CA3AF) |
| `text-slate-500` | 对比度 ~4:1 | 改为 `text-slate-400` (#94A3B8) |
| `text-slate-600` | 对比度 ~3:1 | 改为 `text-slate-400` (#94A3B8) |

### 3. 推荐的深色主题文本层级
```css
/* 主要文本 - 标题、重要内容 */
--text-primary: #F8FAFC;  /* 对比度 ~16:1 */

/* 次要文本 - 正文、描述 */
--text-secondary: #CBD5E1; /* 对比度 ~10:1 */

/* 三级文本 - 标签、时间戳 */
--text-tertiary: #94A3B8;  /* 对比度 ~5.5:1 */

/* 禁用/占位符文本 */
--text-disabled: #64748B;  /* 对比度 ~3.5:1 - 仅用于大文本 */
```
